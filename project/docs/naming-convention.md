# 네이밍 컨벤션

## 파일명 규칙

| 파일 종류 | 규칙 | 예시 |
|-----------|------|------|
| 채널 데이터 | `YYYY-MM-DD_channel.csv` | `2025-01-01_channel.csv` |
| AppsFlyer | `YYYY-MM-DD_appsflyer.csv` | `2025-01-01_appsflyer.csv` |
| 조인 결과 | `YYYY-MM-DD_joined.csv` | `2025-01-01_joined.csv` |

저장 위치:
- 원본 채널: `data/raw/channel/`
- 원본 AF: `data/raw/appsflyer/`
- 조인본: `data/processed/`
- 내보내기: `data/exports/`

---

## 캠페인 네이밍 구조

```
GGL_CMP_01_플러스가입
[채널코드]_CMP_[번호]_[목적]
```

| 채널코드 | 채널 |
|----------|------|
| GGL | 구글 |
| META | 메타 |
| NVR | 네이버 |

---

## 소재 네이밍 구조

```
VID_플러스멤버십_겨울_A_v1
[유형]_[메시지]_[시즌]_[AB]_[버전]
```

| 유형 코드 | 설명 |
|-----------|------|
| VID | 동영상 |
| IMG | 이미지(단일) |
| CRS | 카루셀 |
| TXT | 텍스트(검색) |

| AB 코드 | 설명 |
|---------|------|
| A | 첫 번째 변형 |
| B | 두 번째 변형 |

버전(v1, v2): 동일 소재 리비전

---

## 그룹(타겟) 분류

| 그룹명 | 설명 |
|--------|------|
| 논타겟 | 관심사 기반 신규 |
| 유사타겟 | Lookalike |
| 리마케팅 | 방문자 재타겟 |
| VIP | 기존 구매 고객 |
| 윈백 | 이탈 구매 고객 |

---

## KPI 계산식

| 지표 | 계산식 | 단위 |
|------|--------|------|
| ROAS | 매출 / 비용 × 100 | % |
| CPA | 비용 / 구매건수 | 원 |
| CPC | 비용 / 클릭수 | 원 |
| CTR | 클릭 / 노출 × 100 | % |
| CVR | 구매 / 클릭 × 100 | % |

---

## 데이터 신뢰 우선순위

| 지표 | 우선 소스 | 이유 |
|------|-----------|------|
| 노출·비용 | 채널 데이터 | 플랫폼 원본 |
| 전환·매출 | AppsFlyer | MMP 중복제거 기준 |
| 클릭 | 채널 데이터 | 기본, AF로 보완 |
